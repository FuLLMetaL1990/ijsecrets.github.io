Reveal.js with Org layer

@ejelome

Created: 2019-12-29 Sun 14:52

WHY?

  1. You are already using Spacemacs
  2. You want to use Org mode to create documents
  3. You want to present them as slideshows in HTML

Prerequisites

  1. Spacemacs

    Notes:

    • an Emacs distribution
    • with 18.8k stars on GitHub

  2. Org layer

    Notes:

    • enables Org mode on Spacemacs

  3. reveal.js

    Notes:

    • enables creating slide presentations on HTML

Install Org layer

  1. Open .spacemacs 

    Press: SPC f e d

    Notes:

    • the init file (or initialization file) for Spacemacs
    • in plain Emacs, this is the .emacs, .emacs.el or init.el inside the .emacs.d directory

  2. Add Org layer 

    dotspacemacs-configuration-layers
'(
  ...
  (org :variables
       org-enable-reveal-js-support t)

    Notes:

    • loads the configuration layers during Spacemacs startup or sync

  3. Sync configuration 

    Press: SPC f e R

    Notes:

    • loads the configuration layers without the need to re-open Spacemacs

Setup reveal.js

  1. Download the latest release 

    $ wget https://github.com/hakimel/reveal.js/archive/3.8.0.zip

  2. Unzip Source code 

    $ unzip 3.8.0.zip

  3. Rename directory 

    $ mv reveal.js-3.8.0 reveal.js

    Notes:*

    • ./reveal.js is the default relative path where the org file looks for the reveal.js assets

Create a slideshow

  1. Create an .org file 

    #+TITLE: Hello World!
#+AUTHOR: @username
#+OPTIONS: toc:nil num:nil

* Do this

And this will happen ...

    Notes:

    • #+TITLE along with #+AUTHOR are what appears on the landing page
    • toc:nil removes Table of Contents
    • num:nil removes number in headings
    • * Do this is the heading and will be the title of a slide
    • And this will happen ... is the content of the slide

  2. Export to reveal.js 

    # Export without opening a browser:
Press: C-c C-e v V

# Or export then open default browser:
or press: C-c C-e v b

And that's it! :)
